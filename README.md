# Activation-Based Language Similarity (ABLS)

This repository contains the code for the ACL submission:
"How Do Multilingual LLMs View Language Similarity?
A Perspective from Activation Perturbation"

## Status
This repository contains the core implementation and experiment structure.
Additional scripts and detailed instructions will be released upon acceptance.

## Structure
- `Structure/`: activation perturbation and probing
- `Models_GPD/`, `Models_LPD/`: model wrappers
- `utils/`: shared utilities

## Note
Due to the anonymous review policy, some paths and scripts are simplified.
